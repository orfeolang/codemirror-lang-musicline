# Lang Support CodeMirror

Orfeo and Musicline language support for CodeMirror.

## About

These are lezer grammars for Orfeo and Musicline. They are used
by CodeMirror to highlight code.

## Installation

```sh
npm install
npm run prepare
npm test
```
